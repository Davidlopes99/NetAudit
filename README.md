# NetAudit

Sistema de Auditoria Básica de Serviços em Rede.

## Objetivo

Identificar hosts ativos, portas abertas e classificar riscos básicos de segurança em uma rede.

## Funcionalidades

- Descoberta de hosts
- Varredura de portas TCP
- Identificação de serviços
- Classificação de risco
- Relatório final

## Execução

```bash
python NetAudit.py 192.168.1.1 192.168.1.20
